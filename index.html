Elemento Descripcion Documentacion
Crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL. Documentacion
abbr Representa una abreviación o acrónimo Documentacion
address El elemento address indica que el HTML contenido proporciona información de contacto para una persona o personas, o para una organización. Documentacion
area Sirve para crear las distintas secciones Documentacion
article Representa una composición auto-contenida en un documento, página, una aplicación o en el sitio, que se destina a distribuir de forma independiente o reutilizable, por ejemplo, en la indicación. Documentacion
aside El elemento representa una parte de un documento cuyo contenido solo está indirectamente relacionado con el contenido principal del documento. Documentacion
audio El elemento se utiliza para incrustar contenido de sonido en los documentos. Puede contener una o más fuentes de audio, representadas mediante el atributo src o el elemento source: el navegador elegirá la más adecuada. Documentacion
b Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando. Documentacion
bdi Le dice al algoritmo bidireccional del navegador que trate el texto que contiene de forma aislada del texto que lo rodea. Documentacion
bdo Anula la direccionalidad actual del texto, de modo que el texto que contiene se representa en una dirección diferente. Documentacion
base Especifica la dirección URL base que se utilizará para todas las direcciones URL relativas contenidas dentro de un documento. Documentacion
body Representa el contenido de un documento HTML. Solo puede haber un elemento en un documento. Documentacion
blockquote El elemento blockquote indica que el texto contenido es una cita extensa. Documentacion
br Representa un salto de linea en el documento . Documentacion
button Representa un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar y simple de aplicar. Documentacion
canvas Utilice el elemento HTML canvas con la API de secuencias de comandos de lienzo o la API de WebGL para dibujar gráficos y animaciones. Documentacion
cite El atributo cite especifica la fuente de la cita. El valor del atributo debería ser un enlace al documento original o fuente. Documentacion
code El elemento code muestra su contenido con formato para indicar que el texto es un fragmento corto de código de computadora. Documentacion
data El elemento vincula una determinada pieza de contenido con una traducción legible por máquina. Documentacion
datalist .Contiene un conjunto de elementos option que representan las opciones permitidas o recomendadas disponibles para elegir dentro de otros controles. Documentacion
del El elemento del marca las partes de un texto que han sido eliminadas del documento. Documentacion
details Crea un widget de divulgación en el que la información es visible solo cuando el widget se cambia a un estado "abierto". Documentacion
div Sirve para crear secciones o agrupar contenidos. Documentacion
dfn El elemento dfn se utiliza para identificar el término a definir, mientras que la definición del término se coloca fuera del elemento. Documentacion
dl Representa una lista de descripcion. Documentacion
em Cuando queremos dar énfasis al lenguaje hablado, acentuamos y prolongamos la pronunciación de ciertas palabras y así alteramos sutilmente el significado de lo que decimos. Documentacion
embed Incrusta contenido externo en el punto especificado del documento. Documentacion
fieldset Se utiliza para agrupar varios controles, así como etiquetas dentro de un formulario web. Documentacion
figure Representa contenido autónomo, potencialmente con un título opcional, que se especifica mediante el elemento Documentacion
footer Representa un pie de página para su ancestro más cercano que secciona el contenido o secciona el elemento raíz. Documentacion
form Sirve para representar una seccion de un documento que contiene controles interactivos para poder subir informacion. Documentacion
h1-h6 Los elementos de encabezado implementan seis niveles de encabezado del documento. Documentacion
head Contiene lo que no se ve en la pagina web Documentacion
header Representa un grupo de ayudas introductorias o de navegación. Puede contener algunos elementos de encabezado, así como también un logo, un formulario de búsqueda, un nombre de autor y otros componentes. Documentacion
hr El elemento hr representa una transición entre párrafos a un tema separado, pero relacionado. Documentacion
hgroup Representa un encabezado y contenido relacionado. Documentacion
html Representa la raiz del documento HTML Documentacion
i El elemento representa un rango de texto que se separa del texto normal por alguna razón, como texto idiomático, términos técnicos, designaciones taxonómicas, entre otros. Documentacion
iframe El elemento representa un contexto de navegación anidado, incrustando otra página HTML en la actual. Documentacion
input Se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario. Documentacion
ins El elemento ins marca las partes de un texto que han sido añadidos al documento. Documentacion
img Representa una imagen en el documento. Documentacion
kbd Representa un fragmento de texto en línea que indica la entrada del usuario desde un teclado. Documentacion
label Representa un título para un elemento en una interfaz de usuario. Documentacion
li Es usado para representar un elemento en una lista. Documentacion
link Especifica la relación entre el documento actual y un recurso externo. Documentacion
noscript Aporta contenidos alternativos al elemento script. las aplicaciones de usuario que no soporten scripts deben mostrar en su lugar el contenido de este elemento. Documentacion
main Documentacion
map Se utiliza con elementos area para definir un mapa de imagen (un área de enlace en la que se puede hacer clic). Documentacion
menu Se describe en la especificación HTML como una alternativa semántica a ul, pero los navegadores lo tratan (y lo exponen a través del árbol de accesibilidad) como no diferente de ul. Documentacion
mark El elemento mark permite resaltar texto de interés. Piense en esto como si usara un resaltador en un libro para marcar pasajes que encuentre de interés. Documentacion
meta Sirve para aportar información sobre el documento. Documentacion
nav Representa una seccion de la paguna, la cual tiene el proposito de proporcionar links de navegacion. Documentacion
noscript Define una sección de HTML que se insertará si un tipo de secuencia de comandos en la página no es compatible o si las secuencias de comandos están actualmente desactivadas en el navegador. Documentacion
object Representa un recurso externo, que se puede tratar como una imagen, un contexto de navegación anidado o un recurso para ser manejado por un complemento. Documentacion
ol Representa elementos ordenados en una lista. Documentacion
output Es un elemento contenedor en el que un sitio o aplicación puede inyectar los resultados de un cálculo o el resultado de una acción del usuario. Documentacion
p Este elemento representa un parrafo. Documentacion
picture contiene cero o más elementos source y un elemento img para ofrecer versiones alternativas de una imagen para diferentes escenarios de pantalla/dispositivo. Documentacion
pre Representa texto preformateado. El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo Documentacion
progress Muestra un indicador que muestra el progreso de finalización de una tarea, que normalmente se muestra como una barra de progreso. Documentacion
q El elemento q indica que el texto encerrado es una cita corta en línea. Documentacion
ruby Representa pequeñas anotaciones que se representan encima, debajo o al lado del texto base, generalmente utilizadas para mostrar la pronunciación de los caracteres del este asiático. Documentacion
s El elemento s muestra el texto tachado con una línea que lo atraviesa. Documentacion
samp Se utiliza para encerrar texto que representa la salida de un programa de computadora. Documentacion
section Representa una sección independiente genérica de un documento, que no tiene un elemento semántico más específico para representarlo. Documentacion
select Representa un control que proporciona un menú de opciones. Documentacion
script Se utiliza para incrustar código o datos ejecutables; esto generalmente se usa para incrustar o hacer referencia al código JavaScript. Documentacion
span Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. Documentacion
style Es el elemento encargado de indicar la información de estilo. Documentacion
small El elemento small permite definir la letra chica y los comentarios secundarios de una página, como descargos de responsabilidad, derechos de autor, información sobre licencias y restricciones legales. Documentacion
sub Permite definir subindices. Documentacion
sup Permite definir superindices. Documentacion
strong El elemento strong indica que su contenido tiene una gran importancia, seriedad o urgencia. Los navegadores normalmente muestran el contenido en negrita. Documentacion
table Representa informacion tabular. Representa datos en dos o mas dimensiones Documentacion
tbody Encapsula un conjunto de filas de tabla Documentacion
template Es un mecanismo para contener HTML que no se debe representar inmediatamente cuando se carga una página, pero se puede crear una instancia posteriormente durante el tiempo de ejecución mediante JavaScript. Documentacion
textarea Representa un control de edición de texto sin formato de varias líneas, útil cuando desea permitir que los usuarios ingresen una cantidad considerable de texto de forma libre, por ejemplo, un comentario en una revisión o formulario de comentarios. Documentacion
thead Define un conjunto de filas que definen la cabecera de las columnas de la tabla. Documentacion
time Representa un período específico en el tiempo. Puede incluir el atributo de fecha y hora para traducir las fechas a un formato legible por máquina, lo que permite obtener mejores resultados en los motores de búsqueda o funciones personalizadas, como recordatorios. Documentacion
title Define el titulo del elemtno que se vera en la barra dek navegador. Documentacion
u Representa un tramo de texto en línea que debe representarse de manera que indique que tiene una anotación no textual. Documentacion
ul Representa elementos desordenados en una lista. Documentacion
var Representa el nombre de una variable en una expresión matemática o un contexto de programación. Documentacion
video Incrusta un reproductor multimedia que admite la reproducción de video en el documento. Documentacion
wbr El elemento vacío wbr le indica al navegador donde podría razonablemente insertar un salto de línea para ajustar el contenido que es más grande que la ventana actual del navegador Documentacion
