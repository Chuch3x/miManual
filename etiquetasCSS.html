Colores en CSS

Elemento Descripcion Ejemplo
color Podemos especificar un color usando un nombre predefinido. Por ejemplo podemos especificar que el color del párrafo sea rojo con la siguiente regla: elemento{ color:red; }
rgba Podemos establecer el valor de un color mezclando diferentes intensidades de los tres colores primarios(rojo, verde, azul). elemento{ color: rgb(37, 150, 190); }
rgba Podemos definir la transparencia de un color usando la función rgba, esta función es idéntica a la función rgb, pero con un parámetro adicional denominado opacidad, para especificar la transparencia. elemento{ color: rgba(37, 150, 190, 1); }
Hexadecimal Podemos utilizar un número hexadecimal para definir un color. Es una alternativa a la función rgb, es decir permite definir un color como la mezcla de intensidades de los colores primarios elemento{ color: #numero-hexadecimal; }
hsl En este modelo utilizamos conceptos que son más fáciles de comprender. En CSS para poder trabajar con este modelo tenemos la siguiente opción: elemento{ color: hsl(h, s, l); }
hsla Podemos definir la transparencia de un color usando la función hsla, esta función es idéntica a la función hsl, pero con un parámetro adicional denominado opacidad, para especificar la transparencia. elemento{ color: hsla(h, s, l, a); }

Selectores

Permiten seleccionar elementos del documento a los que queremos aplicar estilos.

Elemento Descripcion Ejemplo
Selector Universal ( * ) Selecciona todos los elementos. *{ declaraciones; }
Selector de Tipo Permite seleccionar elementos usando el nombre del elemento. elemento{ declaraciones; }
Selector de clase (.foo) Permite seleccionar elementos en función de los valores del atributo class de los elementos. Para que los elementos sean seleccionados, su atributo class debe incluir un valor que coincida con el valor del selector. < p class=”foo bar”> ¡HOLA MUNDO! </p> < p class=”foo baz” > ¡ADIÓS MUNDO! </p> .foo{ declaraciones; }
Selector de ID (#foo) Permite seleccionar un elemento en función del valor del atributo ID del elemento. Para que el elemento sea seleccionado, su atributo id debe coincidir exactamente con el valor dado en el selector. #foo{ declaraciones; }
Selector de atributo (a[foo]) Permite seleccionar elementos en función de la presencia o del valor de un determinado atributo. #foo{ declaraciones; }
Selecciona elementos que tengan un atributo con el nombre attr. selector[attr]{ declaraciones; }
Selecciona elementos que tengan un atributo con el nombre attr. El único valor del atributo es exactamente value. selector[attr=value]{ declaraciones; }
selector[attr~=value] { … } Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo es exactamente value. selector[attr~=value]{ declaraciones; }
Selecciona los elementos que tengan un atributo con el nombre attr. El primero de los valores del atributo empieza con value. selector[attr^=value]{ declaraciones; }
Selecciona los elementos que tengan un atributo con el nombre attr. El último de los valores del atributo termina con value. selector[attr$=value]{ declaraciones; }
Selecciona los elementos que tengan un atributo con el nombre attr. Uno de los valores del atributo contiene al menos una ocurrencia de value. selector[attr*=value]{ declaraciones; }

PSEUDO-CLASES

Permite seleccionar elementos en base a su estado (:disabled), posicion de los elementos en el documento(:nth-child) e interacción del usuario(:hover). Ejemplo:

button:hover { color: blue; }
